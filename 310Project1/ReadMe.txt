What 310Project1.py does:
310Project1.py includes two array classes, one implimented as a stack, and one as a queue.  
The stack class includes an extra function called "fromPostFix" that takes an equation in 
postfix notation in the form of a string, and evaluates the expression, printing an integer.
The queue class includes an extra function called "radixSort" that uses radix sort to sort 
and print a given set of two-digit integers.

To use file:  
Open 310Project1.py in PyCharm
Run the script in PyCharm
The data files are included in main

Example usage:
One may simply use the data files in the main, or one may also modify/add to the data files
to give their own input and process their own data
